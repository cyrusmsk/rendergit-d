# rendergit-d
Render any git repo into a single static HTML page for humans or LLMs

## Build 
Just run:
```bash
dub build -b=release
```

## Usage
To find how to use the CLI run:
```bash
./rendergit -h
```

## Reference
Original [implementation](https://github.com/karpathy/rendergit) in Python by Karpathy
